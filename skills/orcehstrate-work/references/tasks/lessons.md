# Workflow Lessons

Record durable, evidence-backed process improvements at campaign boundaries.
Do not use this file as an execution log.

## Current Defaults

- Focus review on correctness, breakage, security, architecture,
  maintainability, performance, documentation accuracy, and principle conflicts.
- Avoid style feedback unless it creates material risk.
- Use risk-based gates instead of every reviewer for every task.
- Keep worker contexts scoped and black-box review isolated.
